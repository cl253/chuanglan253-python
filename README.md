## chuanglan

**chuanglan-demo-python**

## 使用说明

- 下载python2.x版本，本demo仅支持python2.x版本

- 打开sms.py文件里面

- host填写：域名 例：smssh1.253.com


## 源码说明 

- 编码要求为utf-8,请先将编辑器底层语言设置为utf-8

- 带有特殊字符的内容无法直接提交,需先将特殊字符进行urlencode编码后方能提交

- 开发API可参考单元测试 doc/253云通讯PaaS短信云接口说明（JSON版）.docx
